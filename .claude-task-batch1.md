# Task Batch 1: 文件上传 UI + AI 讲解功能

## 项目根目录
`D:\AI_Project\Surprise\Interview-Practice-App`

## Python 环境
`D:\AI_Envs\pytorch_gpu\python.exe`

## 背景
后端 API 已全部实现：
- `POST /api/v1/files/upload` — 文件上传 + 解析 + 题目提取
- `POST /api/v1/ai/explain` — AI 题目讲解（depth: brief/standard/deep）
- `GET /api/v1/questions/:id` — 题目详情

前端 API 代理已配置：`/api/*` → `http://localhost:8000/api/*`
使用 axios + TypeScript + Tailwind + Next.js 14 App Router

## 任务 1：导入页添加文件上传组件

修改 `web/src/app/import/page.tsx`，在现有文本导入区域下方添加文件上传区域：

1. 支持拖拽上传 + 点击选择文件
2. 支持文件类型：PDF, DOCX, TXT, MD
3. 上传时显示 loading 状态
4. 上传成功后显示：文件名、解析状态、提取题目数、知识节点数
5. 提供"查看题目列表"链接

调用 API：
```
POST /api/v1/files/upload
Content-Type: multipart/form-data
Body: { file: File }
```

响应格式：
```json
{
  "status": "success",
  "file_id": "uuid",
  "file_name": "xxx.pdf",
  "parse_status": "success",
  "questions_extracted": 5,
  "knowledge_nodes": 12,
  "question_ids": ["uuid1", "uuid2"]
}
```

## 任务 2：题目详情页添加"生成讲解"按钮

修改 `web/src/app/questions/[id]/page.tsx`：

1. 在题目内容下方添加讲解按钮组：brief / standard / deep
2. 点击按钮调用 `POST /api/v1/ai/explain`，body: `{ question_id, depth }`
3. 响应：`{ answer_short, answer_detail, explanation, depth }`
4. 讲解内容用现有的 markdown 组件渲染
5. 如果题目已有 `answer_detail` 字段（非空），默认展开显示
