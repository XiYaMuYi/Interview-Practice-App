# Done: Fix Streaming Proxy Bugs

## Completed Fixes

### 1. Frontend `/api/v1/ai/explain-stream` proxy 502
- **Removed** `web/src/app/api/v1/ai/explain/route.ts` — dead code proxy that could conflict with `explain-stream` in Next.js routing
- **Hardened** all three AI proxy routes (`explain-stream`, `turn-stream`, `[...path]`) with:
  - 30s timeout (`AbortController`)
  - `Content-Type: application/json` header injection when missing
  - `transfer-encoding` added to stripped headers
  - Empty body fallback to `"{}"`
  - Better error classification (timeout vs connection refused vs generic)
  - Exposed `code` field in error response for debugging

### 2. `interview/turn-stream` frontend-backend alignment
- **Cleaned up** `web/src/app/interview/page.tsx` `startInterview`:

- **Fixed** `InterviewStartResponse` TypeScript interface to match backend (removed phantom `domain` and `difficulty` fields)
