# 前端 Token 级流式对接

## 背景
后端已经完成了 token 级真流式改造，SSE 事件新增了 `token` 类型：
```
event: token
data: {"task_id": "...", "token": "逐字文本片段"}
```

后端 stream 端点：
- `POST /api/v1/ai/explain-stream` → 返回 (task_id, SSE stream)，有 `token` 事件
- `POST /api/v1/ai/interview/turn-stream` → 返回 (task_id, SSE stream)，summary 阶段有 `token` 事件
- `POST /api/v1/ai/review/summary-stream` → 返回 (task_id, SSE stream)，有 `token` 事件

## 需要改造的文件

### 1. `src/hooks/useTaskEvents.ts` — 新增 token 累积

**现状**: `TaskEvent` 接口没有 `token` 字段，hook 不处理 `token` 事件。

**改动**:
- `TaskEvent` 接口新增 `token?: string` 字段
- `UseTaskEventsReturn` 新增 `accumulatedContent: string`（累积的完整文本）
- 在 SSE 解析循环中，当 `eventType === 'token'` 时：
  - 将 `data.token` 追加到 `accumulatedContent`
  - 同时保留原有的事件收集逻辑不变

### 2. `src/app/questions/[id]/page.tsx` — 题目详情页改用流式讲解

**现状**: `handleGenerate()` 函数用 `axios.post('/api/v1/ai/explain', ...)` 调用同步接口，阻塞等待完整结果。

**改动**:
- 新增 `handleGenerateStream(depth)` 函数：
  1. 调用 `POST /api/v1/ai/explain-stream`，拿到 `task_id`
  2. 使用 `useTaskEvents(taskId)` hook 消费 SSE 流
  3. 用 `accumulatedContent` 做打字机效果渲染
  4. `done` 事件触发时保存最终结果
- UI 改动：在进度条下方增加一个"实时输出"区域，流式渲染累积文本（用 `<pre>` 或 `<div>` + markdown）
- 原有的同步 `handleGenerate` 保留作为 fallback

### 3. `src/app/exam/session/[id]/page.tsx` — 面试会话页改用流式

**现状**: 使用 `EventSource` 监听流，但没有处理 `token` 事件。

**改动**:
- 新增 `token` 事件处理，累积面试总结文本
- 实时显示面试总结的生成过程（打字机效果）

## 技术细节

### SSE 流调用模式
已有的后端 API 返回格式：
```ts
// 第一步：POST 获取 task_id
const res = await fetch('/api/v1/ai/explain-stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ question_id: id, depth: 'standard' }),
});
const data = await res.json();
const taskId = data.task_id;

// 第二步：通过 /api/v1/tasks/{taskId}/events 获取 SSE 流
// （useTaskEvents hook 已经处理了这个端点）
```

### 打字机效果
- 在组件中添加一个状态：`const [streamingText, setStreamingText] = useState('')`
- 使用 `useTaskEvents` 的 `accumulatedContent` 更新 `streamingText`
- 用 `<div className="whitespace-pre-wrap">` 或现有的 `<MarkdownContent>` 渲染

### 样式要求
- 使用已有的 `soft-card` 样式容器
- 流式输出区域用灰色背景区分
- 加载中显示进度条 + 当前阶段
- 完成时显示完整 markdown 渲染结果
